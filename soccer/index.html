La Liga ML Predictions

This application predicts La Liga football outcomes and simulates league standings. It trains machine learning models (RandomForest, XGBoost, and LightGBM) using historical match data and team statistics, which includes estimated expected goals (xG). It also processes data from multiple past seasons (20/21-23/24) to predict outcomes for future seasons (24/25-25/26). For the 25/26 season, it uses "real" standings data.

Match and team data sourced from FBref.com, courtesy of Sports Reference LLC, accessed via the soccerdata Python package. Custom schedule and standings data compiled by the author. Modeling powered by scikit-learn, XGBoost, and LightGBM.

Data Sources: Match and team data from FBref.com via soccerdata Python package

Technologies: scikit-learn, XGBoost, LightGBM
